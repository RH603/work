.red라는 클라스를 사용하면 바탕화면이 붉은색이 됩니다
```htnl
<p class="red"> 붉은색이 됩니다.</p>
```

css 내용은 아래와 같습니다.

```css
.red{
    background-color:red;
}
```

absolute position을 이용해서 중앙 정렬하는 방법
```css
.center{
    position:absolute;
    top:50%;
    left:50%;
    transform:translate(-50%, -50%);
}
```
